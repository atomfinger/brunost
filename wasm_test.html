Brunost

Programmeringsspråket med smak av Noreg

Brunost er eit programmeringsspråk designa for dei som meiner at kode ikkje berre skal vere funksjonell — han skal ha sjel. Skriv med ekte nynorsk terminologi: forKvart, gjevTilbake, prøv og fang. Kode som flyt like mjukt som brunost over ein nysteikt vaffel. Du kan lese meir om Brunost på GitHub

Variablar

fast namn er "Noreg" endreleg tal er 0

Løkker

forKvart x i liste { } medan (vilkår) gjer { }

Funksjonar

gjer namn(a, b) {   gjevTilbake a+b }

Feilhandtering

prøv {   kast "feil" } fang (e) { }
